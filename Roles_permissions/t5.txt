

1. Register Company A

API

POST /api/register/company/

Body


{
    "company_name": "ABC Software",
    "description": "Software Development Company",
    "username": "Faisal",
    "email": "faisal@gmail.com",
    "password": "12345678"
}


Response


{
    "message": "Company created successfully.",
    "company": "ABC Software"
}


Creates the first company along with its owner account.

9. Access Another Company's Project

API

GET /api_project/1

Response

{
    "detail": "No Project matches the given query."
}


Confirms that users cannot access a project owned by another company, even if they know its ID.
